
# 📝 To-Do App (Task 3)

🚀 This is a Full Stack To-Do Application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

It helps users manage daily tasks efficiently with simple and clean UI.

---

## ✨ Features:

* ➕ Add new tasks
* 📝 Update/Edit tasks
* ❌ Delete tasks
* 📋 View all tasks
* ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack:

* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** MongoDB

---

## 📌 Purpose:

This project is created as part of Task 3 to practice full stack development and understand CRUD operations.

---

## ⚙️ How to Run:

1. Clone the repository
2. Install dependencies:

   ```
   npm install
   ```
3. Start backend:

   ```
   node server.js
   ```
4. Start frontend:

   ```
   npm start
   ```
